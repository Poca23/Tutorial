Les six titres HTML différents :

Définition et utilisation :
Les balises < h1> à < h6> sont utilisées pour définir les titres HTML.
Remarque : N'en utilisez qu'un < h1> par page ; cela doit représenter le titre/sujet principal de la page entière. Ne sautez pas non plus les niveaux de titre : commencez par < h1>, puis utilisez < h2>, et ainsi de suite.

Prise en charge :

Attributs globaux :
Les balises < h1>to < h6>prennent également en charge les attributs globaux en HTML.

Attributs d'événement :
Les balises < h1>to < h6> prennent également en charge les attributs d'événement en HTML.
